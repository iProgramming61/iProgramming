Data Science in Python Machine Learning & AI Summer Program 1

Our popular introductory summer program covers mastering Python functions, analyzing big data, and building machine learning algorithms. Students are grouped in cohorts based on experience and age. Everyone completes a function challenge, and presents a portfolio project using big data and machine learning. For students new to data science, and first-time coders.

Deep Learning in Python Machine Learning & AI Summer Program 2

This exciting new summer program covers classifying images, predicting text, and recommending books, movies, or music. Students build and fine-tune neural networks using keras and TensorFlow. For students who have already taken a machine learning class, or for advanced Python programmers who can fill in gaps on their own.
